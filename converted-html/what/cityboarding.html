Cityboarding

2013, London. Prototyping and backend design.

Arduino, Processing, Java

Collaborators: Giuseppe Burdo, Stefano Trento, Stefano Baldan

Cityboarding is an iphone app connected to a 3D printed bus which allows you to know when and from where to board.

Started as an Arduino hack, we worked on this project to explore new way to interact with the London public transportations system, personal spaces like the house, offices, schools and physical and portable objects.

Abandoned due to lack of time, the project evolved into Superclock.
